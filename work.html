Clearing Feed Generation is a web application system that generates a feed file towards a clearing system in a specific format based on the input transactions to the system. Monitoring the lifecycle of the data particularly the transformations applied to the data in the process are captured using sql statement data lineage. This project primarily used Hadoop, SparkSql, Spline (SPark LINEage) and MongoDB.
